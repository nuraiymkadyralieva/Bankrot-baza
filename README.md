# Bankrot-Baza Parser

Инструмент для автоматизации сбора открытых данных о торгах по банкротству с портала bankrotbaza.ru. Программа использует браузерную автоматизацию для навигации по сайту, обработки детальной информации о лотах и выгрузки сведений в формат Excel.

## Основной функционал

Программа реализует полный цикл автоматизированного сбора данных:
* **Автоматизация навигации**: Последовательный переход по страницам поисковой выдачи по заданному адресу.
* **Обработка детальных карточек**: Переход по прямым ссылкам в каждый лот для извлечения характеристик, недоступных в общем списке.
* **Сбор сопутствующих сведений**: Переход на связанные страницы должников для получения ИНН, ОГРН, региона и номера судебного дела.
* **Экспорт в Excel**: Формирование файла .xlsx с автоматической настройкой ширины колонок и базовой очисткой данных.

## Технические решения и логика

### Взаимодействие с интерфейсом и авторизация
Для корректного отображения контента в парсере предусмотрены следующие механизмы:
* **Учёт пользовательских проверок**: В коде задан таймаут до 90 секунд (HUMAN_CHECK_TIMEOUT_SEC), позволяющий оператору при необходимости завершить проверку пользователя в окне браузера до продолжения работы алгоритма.
* **Ожидание авторизации**: Для обеспечения доступа к разделу документов реализован метод waitForLoginIfNeeded. Программа ожидает выполнения входа в личный кабинет до 120 секунд, ориентируясь на появление элементов профиля в коде страницы.
* **Регулировка интенсивности**: Между запросами к отдельным лотам установлена задержка в 1200 мс (SLEEP_BETWEEN_LOTS_MS) для снижения частоты обращений к серверу.
  
* **Подготовка данных**: При записи в таблицу метод clean приводит значения к единообразному виду, исключая технические артефакты (null, -, н/д) и оставляя ячейки пустыми согласно ТЗ.

## Используемые технологии
* **Java 17**
* **Selenium WebDriver**: Автоматизация управления браузером Chrome.
* **Apache POI (XSSF)**: Работа с форматом Excel.
* **WebDriverManager**: Настройка окружения для запуска драйвера.

## Запуск проекта
1. Убедитесь в наличии установленного браузера Google Chrome.
2. Выполните сборку проекта.
3. Запустите класс Main.java. Программа инициализирует браузер, перейдет по ссылке и предоставит время для авторизации, после чего начнет сбор сведений.
